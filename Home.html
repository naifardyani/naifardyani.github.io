Tentang

Saya adalah seorang pelajar SMP yang sedang belajar mengenai web development. Saya belajar mengenai dunia web untuk menghindari kebosanan ketika masa pandemi Covid-19, tapi saya malah kecanduan dan belajar lebih lanjut tentang dunia web. Memiliki impian untuk menjadi seorang freelancer dalam bidang web development karena lebih santai dan fleksibel. Saya juga sedang belajar bahasa Inggris untuk memperdalam ilmu saya.

Sekarang saya hanya bisa melakukan HTML dan CSS, dan web ini dibuat dengan HTML dan CSS serta dengan bantuan Bootstrap. Namun saya berencana untuk belajar Javascript, PHP, Python, dan C++ di masa depan. Tidak lupa, saya juga sedang belajar mengenai desain dasar agar website saya dapat terus berkembang lebih baik. Sebenarnya ini adalah website pertama saya yang dikerjakan dengan serius ketika saya kelas 7. Terima kasih juga telah datang di website saya.
